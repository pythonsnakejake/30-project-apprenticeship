   - Why is using Python's `with open("file.csv", "r") as file:` framework (Context Manager) considered far safer and superior to simply using

   with open() closes the file after we have finished using it, whereas file =  open() does not, we have to close it ourselves. If we forget to close

We can stream our file line-by-line using a file iterator loop. To skip the header line cleanly without loading the whole file into memory, 
we can use Python's native next() function before the loop starts.

```
with open("shipping_manifest.csv", "r") as file:
    header = next(file)  # Streams and consumes exactly ONE line (the header), advancing the cursor!
    for line in file:    # Automatically starts streaming from row 2 onwards!
```

This streaming strategy keeps our memory footprint at zero, whether the file has 3 rows or 30,000,000 rows.
